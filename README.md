This repository contains the CSA15 Cloud Computing and Big Data Analytics Assessment Tool 1 (CO1) submission. It includes four concept maps created using diagrams.net that illustrate the fundamental concepts of cloud computing. The maps cover Cloud Computing Fundamentals, Evolution of Cloud Computing, Cloud Service Models (IaaS, PaaS, SaaS, and XaaS), and Virtualization & Web Services as the foundation of cloud computing. Each concept map demonstrates the relationships between cloud characteristics, deployment and service models, scalability, elasticity, resource pooling, APIs, REST, SOAP, virtualization, and cloud architecture using hierarchical organization and labeled connections. The repository also contains the editable .drawio files, exported PNG images, assignment documentation, and supporting files required for submission. This project demonstrates the application of cloud computing principles and fulfills Course Outcome CO1 by presenting cloud technologies in a structured, logical, and visually understandable format.
